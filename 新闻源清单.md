# 特殊教育行业日报｜新闻源清单

最后更新：2026-05-25  
用途：作为《特殊教育行业日报》的默认信源白名单、搜索优先级与数据核验依据。生成日报时，优先按本文件查找、核验和引用来源。

## 使用规则

1. 一级信源用于确认政策、金额、人数、机构动态、认证规则、研究结论和原始数据。
2. 二级信源用于发现线索、补充背景、追踪行业趋势，重要事实必须回到一级信源核验。
3. 三级信源用于观察家长需求、舆论变化、招聘扩张和内容热度，不单独作为关键事实定稿依据。
4. 如果一级与二级冲突，以一级为准；如果官方来源之间冲突，保留不同口径并说明采用哪一口径。
5. 涉及数字、融资、政策、研究结论、认证规则时，必须标注来源链接、发布日期或数据口径。
6. 无法核验的市场传闻、机构宣传或社媒热帖，日报中统一写“暂无可靠数据”或标注为“舆情线索”。
7. `0. 今日一句话总结` 必须只有一句突出主结论；补充观察另起一行，使用弱化说明，不要把多条判断塞进同一段。

## 一级信源｜官方发布 / 原始数据 / 高权威源

适用：政策确认、补贴标准、人数数据、认证规则、研究结论、企业原始动态、融资原始公告。

## 三级信源｜社交平台 / 招聘 / 聚合 / 舆情观察

适用：找线索、判断家长需求、观察舆论与招聘扩张。不得单独作为关键事实或数据依据。

### 1. 国内社交与内容平台

- 微信公众号 / 视频号：重点跟踪机构官方号、医生科普号、家长社群号。
- 小红书：家长真实需求、康复费用、融合教育、机构口碑、AI 工具体验。
- 抖音：机构营销、公益传播、家长咨询热点、短视频流量变化。
- 微博：公共事件、政策讨论、公益议题、舆论风险。
- 知乎：ABA、感统、融合教育、诊断与干预争议。
- B站：家长经验、科普长视频、机构课程传播。
- 第 6 节引用社交平台时必须记录：平台、搜索词、检索时间、样本口径、代表性限制。除非平台公开提供可核验统计，否则不得写成“多数家长”“明显上升”等统计化结论。
- 第 6 节优先用可公开访问的媒体报道、政府/公益组织公告、平台话题页或搜索页作为链接证据；私域社群只能写“家长社群观察”，不能作为可引用数据源。

### 2. 国际社交与社区

- LinkedIn：国际机构招聘、产品发布、行业从业者观点。
- X：研究机构、政策组织、公司创始人、学者动态。
- Reddit：家长与从业者讨论，只作舆情线索。
- YouTube / TikTok：海外机构传播、家长内容、公益活动。

### 3. 招聘与扩张观察

- 企业官方招聘页：优先级最高，用于确认岗位、城市、部门和扩张方向。
- BOSS直聘 / 猎聘 / 智联招聘 / 前程无忧：国内岗位数量、城市、薪酬区间、校区招聘。
- LinkedIn Jobs / Indeed / Glassdoor：国际岗位、BCBA/RBT 需求、远程岗位。
- App Store / 应用宝 / 小程序更新：产品迭代、功能变化、用户入口变化。
- 第 9 节必须尽量输出具体岗位和招聘信号，不只写“招聘方向”。推荐字段：`机构｜岗位｜城市/地区｜经验/学历｜扩张信号｜判断｜来源平台/链接`，其中来源列统一放在最后一列；主表不设置独立的“薪资区间”列或“岗位数/样本数”列。
- 如需保留岗位数量、工资样本量等证据，压缩写入“扩张信号”或“判断”字段，不单独成列。
- 第 9.1 小标题固定为 `重点机构岗位`，不要写成 `重点机构岗位与薪资` 或 `重点机构岗位与薪资样本`；薪资相关内容放在 `9.2 行业薪资基准`。
- 薪资口径必须标明来源：招聘平台单岗位薪资、企业招聘页未披露薪资、职友集/Glassdoor/Indeed 聚合样本。聚合薪资不能等同于公司当前所有岗位薪资。
- 如招聘平台只显示岗位但不显示薪资，写“未披露”；如只有搜索结果摘要可见，写“平台公开摘要，需二次点击核验”。
- 第 9 节不要在标题下输出独立的“数据口径”说明段；必要的口径信息放进表格字段、来源列或判断句里，保持页面直接进入岗位与薪资信息。
- 每天优先观察岗位变化：一线康复师、ST/OT/PT、ABA 治疗师、BCBA/BCaBA、督导、评估师、校区负责人、课程顾问、AI 产品/数据/运营岗。

## 每日默认搜索顺序

1. 先查一级信源：教育部、中国残联、卫健委、地方教育局/残联、BACB、CDC、NIH、Autism Speaks、公司官网与公众号。
2. 再查二级信源：用专业媒体和主流报道发现当天候选事件。
3. 对融资、并购、认证、政策、研究结论，回到一级源确认数字与原文。
4. 最后查三级信源：补充家长需求、舆论变化、招聘扩张、流量趋势。
5. 每条重要事实进入日报前，至少记录一个来源链接；核心判断进入 `0.1 今日核心数据与结论依据` 表。
